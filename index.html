استخدم JSONBin.io المجاني لحفظ البيانات على الإنترنت بحيث يراها الجميع.

📖 طريقة الإعداد
1. سجّل مجاناً على jsonbin.io
2. API Keys ← انسخ Master Key
3. أنشئ Bin جديد → {"nodes":{},"nextId":1} → انسخ ID → البيانات الرئيسية
4. أنشئ Bin ثاني → {"tempPws":[]} → انسخ ID → الإعدادات
5. في إعدادات كل Bin → اجعل Private: OFF
